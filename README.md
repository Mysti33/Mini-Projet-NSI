# Projet Web NSI

Projet réalisé dans le cadre de la spécialité NSI ayant pour objectif de découvrir la création d'un site web en utilisant les technologies du web.

## 🎯 Objectif du projet

Créer un site web permettant de mettre en pratique les bases du développement web :
- structure d'une page avec HTML ;
- mise en forme avec CSS ;
- interactions avec JavaScript.

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript

## ✨ Fonctionnalités principales

- Création d'une interface personnalisée ;
- Animations CSS ;
- Mise en place de formulaires ;
- Ajout d'interactions dynamiques avec JavaScript.

## ⚠️ Informations importantes

Les formulaires présents sur le site sont uniquement des démonstrations.
Aucune donnée utilisateur n'est enregistrée, transmise ou exploitée.


## 📚 Compétences développées

Ce projet m'a permis de découvrir :
- les bases du développement front-end ;
- l'organisation d'un projet web ;
- la séparation entre structure (HTML), style (CSS) et logique (JavaScript).
